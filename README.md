# Vcell_State
A repo that can easily use Arc-STATE on Anndata (Cell gene Expression and perturbation) predict. 


Before using this repo, please read [State](https://github.com/ArcInstitute/state?tab=readme-ov-file), [cell-eval](https://github.com/ArcInstitute/cell-eval) and [cell-load](https://github.com/ArcInstitute/cell-load) first. And then install uv tools.


Codes in this repo is for sc gene perturbation dataset, mainly for Replogle-Nadig.You can train, infer and predict model through .sh files and read data, training process and predict results through .ipynb files.


In case of using pre-train models, [State](https://github.com/ArcInstitute/state?tab=readme-ov-file) provides colabs to download ST and SE models.
